# Documentation Overview

Welcome to the Memory-OS documentation. This directory contains detailed guides, architectural decisions, and references for developers working on the codebase.

## Directory Structure

* **[`architecture/`](architecture/overview.md)**: High-level system design, data flows, and architectural decisions.
* **[`setup/`](setup/installation.md)**: Installation, environment configuration, and local development setup.
* **[`features/`](features/)**: Documentation for core product features (Multi-Source Sync, GraphRAG Extraction, Hybrid Search).
* **[`cli/`](cli/)**: How to use and interact with the CLI tool.
* **[`core/`](core/)**: Deep dives into the core engine systems (Pipeline, Databases, LLM Extractors).
* **[`integrations/`](integrations/)**: Information on data source connectors (GitHub, Gmail, Notion, Google Calendar).
* **[`development/`](development/)**: Testing, debugging, and coding conventions.
* **[`reference/`](reference/)**: Ontologies (Entities and Relationships) and other glossaries.
